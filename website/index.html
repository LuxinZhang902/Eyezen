Developer install (unpacked)

  1. Clone or download the EyeZen repo.
  2. Run npm install and npm run build (or npm run build:no-bg).
  3. Open chrome://extensions in Chrome.
  4. Enable Developer mode.
  5. Click Load unpacked and select the dist/ folder.
  6. Pin EyeZen to your toolbar and open the popup.

Tip: After saving color settings, EyeZen injects a gentle tint to the current tab using scripting permissions.
