🔍 Eye Strain Detection Test

📋 Testing Instructions

Step 1: Load the Chrome extension in Developer Mode
  • Open Chrome and go to chrome://extensions/
  • Enable "Developer mode" (top right toggle)
  • Click "Load unpacked" and select the extension folder
Step 2: Open Developer Console
  • Press F12 or right-click → "Inspect"
  • Go to the "Console" tab
  • This is where you'll see the debugging output
Step 3: Activate the Extension
  • Click the extension icon in the toolbar
  • Start the eye tracking feature
  • Position your face in front of the camera
Step 4: Monitor Console Output
  • Look for messages starting with 👁️, 😴, 👀, 🧍, 🔍, 📊, and 🎯
  • These show the real-time eye metrics and calculations

🎯 What to Look For in Console:

  • 👁️ Raw EAR Value: Should be around 0.2-0.4 (higher = more open eyes)
  • 😴 PERCLOS Value: Percentage of time eyes are closed (higher = more fatigue)
  • 👀 Blink Detection: Shows blink rate in blinks per minute
  • 🔍 Fatigue Calculation Input: All the raw metrics being used
  • 📊 Individual Scores: How each metric contributes to fatigue
  • 🎯 Final Fatigue Index: The overall strain score (0-100)

⚠️ Troubleshooting:

  • If no console messages appear, the face detection might not be working
  • Make sure your camera is enabled and face is well-lit
  • The detection now runs every 2 minutes instead of real-time
  • Try blinking slowly or closing your eyes to see if EAR values change
Step 5: Test Eye Strain Scenarios
  • Normal State: Keep eyes normally open - EAR should be ~0.3
  • Tired Eyes: Squint or partially close eyes - EAR should decrease
  • Blinking: Blink normally - should detect blinks and calculate rate
  • Head Position: Lean forward or tilt head - should affect posture score
