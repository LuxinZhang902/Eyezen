This page will help you test if Chrome AI (Gemini Nano) is available and properly configured.

üìã Setup Instructions

Before testing, you need to enable Chrome AI flags:

  1. Open a new Chrome tab and go to: chrome://flags/#optimization-guide-on-device-model
  2. Set it to: "Enabled BypassPerfRequirement"
  3. Go to: chrome://flags/#prompt-api-for-gemini-nano
  4. Set it to: "Enabled"
  5. Restart Chrome completely
  6. Come back to this page and click "Test Chrome AI"

Hardware Requirements:

  • macOS 13+ (Ventura or later)
  • At least 22 GB free disk space
  • GPU with more than 4 GB VRAM
  • Unlimited/unmetered internet connection
